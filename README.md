MPHPC
=====

A modern PHP framework core.

This is my own take on what a modern PHP framework core should be.

Feature List
------------

-   Recursive SASS Compilation
-   Recursive LESS Compilation
-   Recursive JavaScript Compilation
-   Dart Compilation
-   Global Site Settings
-   Database Class
-   Forms Class
